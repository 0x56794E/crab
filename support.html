Support¶

There are several ways to get in touch with the developers:

Mailing List¶

Main mailing list is scikit-crab. There is also our twitter for commits list, where updates to the main repo get notified: @muricoca_crab

Bug tracker¶

If you think you’ve encoutered a bug, please report it to the issue tracker:

https://github.com/muricoca/crab/issues

IRC¶

Some devs like to hang out on channel #muricoca on irc.freenode.net

If you do not have an irc client or are behind a firewall this web client works fine: http://webchat.freenode.net
